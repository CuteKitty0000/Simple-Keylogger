# Simple Keylogger

A minimalistic Python keylogger script using the `pynput` library.

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/yourusername/simple-keylogger.git
    cd simple-keylogger
    ```

2. Install the `pynput` library:

    ```sh
    pip install pynput
    ```

## Usage

Run the script:

```sh
python keylogger.py
